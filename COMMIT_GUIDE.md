## 🚀 核心提交类型

| 类型 | Emoji | 使用场景 | 示例 |
| :--- | :---: | :--- | :--- |
| **note** | 📝 | 每日学习笔记 | `📝 note: 2025-02-28 Go并发模型学习` |
| **review** | 🔁 | 八股文复习整理 | `🔁 review: TCP/IP协议栈整理` |
| **organize** | 🗂️ | 知识体系整理 | `🗂️ organize: 重构算法知识体系` |
| **interview**| 💼 | 面试题整理/复盘 | `💼 interview: 整理Redis高频面试题` |

| **chore** | 🧹 | 仓库目录整理维护 | `🧹 chore: 整理Interview目录结构` |

## 📅 复盘与节奏 (Rhythm)

为了保持“日日寸进”的状态，我需要建立稳定的复盘节奏：

### 🔄 每周例行 (Weekly)
- **周六/日晚上**：总结本周学习日志，制定下周“进击计划”。

### 📅 每月例行 (Monthly)
- **月末周末**：整理当月笔记，归类归档到对应目录。
- **月初第一周**：更新 [Roadmap.md](file:///d:/OneDrive/Desktop/Notes/Byte%20With%20Me/Roadmap.md)，根据实际进度调整重点。
- **专项攻克**：每月至少进行一次系统性的专题复习。

---

## 🏔️ 关键节点规划 (Milestones)

在每个大的时间节点，我需要跳出细节，重新审视“远方”：

### 🎓 学期调整
> 开启新学期前，必须重新调整长期规划。
- [ ] **大二下**：稳扎稳打，深入中间件
- [ ] **大三上**：项目实战，开始备战实习
- [ ] **大三下**：进击暑期实习 Offer
- [ ] **大四上**：秋招决战
- [ ] **大四下**：毕业起航

### ❄️🔥 寒暑假突围
- **假日前期**：复盘上学期得失。
- **假日中期**：高强度攻克硬核技术栈。
- **假日后期**：项目打磨与文档沉淀。
